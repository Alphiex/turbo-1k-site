# Sample Automation Template (Demo)

## Workflow: Trend-to-Offer Content Loop

### Goal
Turn daily AI/tool trend signals into revenue-focused posts and offer traffic.

### Inputs
- 5 trending stories from trusted sources
- 3 social pulse snippets
- Current offer link with tracked source params

### Steps
1. Summarize each signal into one practical takeaway.
2. Generate 3 post variants (X/LinkedIn/Reddit style).
3. Attach the relevant tracked offer URL.
4. Log published post + timestamp + channel.
5. Review clicks/leads at 24h and rank performance.

### KPI Targets
- CTR >= 2.0%
- Lead conversion >= 3.0%
- First sale target <= 7 days

### Output Schema
- channel
- post_copy
- source_url
- publish_time
- clicks_24h
- leads_24h
- notes
